# ainama-react-cli

React template. Customize.

server: node.js

style: sass

js: react-router-dom react-redux lazy()

plugins: code-splitting(js/css)

### Use
```js

sudo npm install ainama-react-cli

cd <your path>

ainama-react-cli init <name>

cd <name>

npm run start

```
